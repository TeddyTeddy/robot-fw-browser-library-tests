16"""A module to handle different character widths on the console. 

17 

18Some East Asian characters have width of two on console, and combining 

19characters themselves take no extra space. 

20 

21See issue 604 [1] for more details about East Asian characters. The issue also 

22contains `generate_wild_chars.py` script that was originally used to create 

23`_EAST_ASIAN_WILD_CHARS` mapping. An updated version of the script is attached 

24to issue 1096. Big thanks for xieyanbo for the script and the original patch. 

25 

26Note that Python's `unicodedata` module is not used here because importing 

27it takes several seconds on Jython. 

28 

29[1] https://github.com/robotframework/robotframework/issues/604 

30[2] https://github.com/robotframework/robotframework/issues/1096 

31"""

33def get_char_width(char): 

34 char = ord(char) 

35 if _char_in_map(char, _COMBINING_CHARS): 

36 return 0 

37 if _char_in_map(char, _EAST_ASIAN_WILD_CHARS): 

38 return 2 

39 return 1 

40 

41def _char_in_map(char, map): 

42 for begin, end in map: 

43 if char < begin: 

44 break 

45 if begin <= char <= end: 

46 return True 

47 return False 

48
